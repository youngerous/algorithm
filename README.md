## Snippet
- ```input()``` 함수보다 ```sys.stdin.readline()```이 빠르다. 
- 문자열을 입력받을 때 ```.rstrip()``` 함수를 통해 마지막 개행문자를 지울 필요가 있다.
- 줄 별로 구분되어 있는 입력 읽어들일 때: ```sys.stdin.read().splitlines()```


```python
import sys
input = sys.stdin.readline
print = sys.stdout.write
```
